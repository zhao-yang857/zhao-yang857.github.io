垂直居中总结

2021-01-12
- -

第一种

给父盒子设置flex布局

.box{
    width:600px;
    height: 200px;
    background: pink;

    display:flex;
    justify-content:center;
    align-items:center;
}

第二种

给父盒子设置text-align属性 设置行高 子元素改为行内元素 多适用于文字居中

.box{
    width:600px;
    height: 200px;
    background: pink;

    text-align: center;
    line-height: 200px;
}
.son{
    width: 100px;
    height: 100px;
    background: yellow;

    display:inline;
}

第三种

设置定位—— 给父盒子设置相对定位 子元素绝对定位+transform 

.box{
    width:600px;
    height: 200px;
    background: pink;

    position: relative;
}
.son{
    width: 100px;
    height: 100px;
    background: yellow;

    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%,-50%);
}

第四种

设置定位—— 给父盒子设置相对定位 子元素绝对定位+设置边距

.box{
    width:600px;
    height: 200px;
    background: pink;

    position: relative;
}
.son{
    width: 100px;
    height: 100px;
    background: yellow;

    position: absolute;
    margin: auto;
    top: 0;
    left: 0;
    bottom: 0;
    right: 0;
}

第五种

padding填充

.box{
    width:600px;
    background: pink;

    padding:100px;
}
.son{
    width: 100px;
    background: yellow;
}
