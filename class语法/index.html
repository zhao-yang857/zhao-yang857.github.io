class基本语法

生成实例对象的传统方法是通过构造函数

function A(x, y) {
  this.x = x;
  this.y = y;
}

A.prototype.toString = function () {
  return '(' + this.x + ', ' + this.y + ')';
};

var p = new A(1, 2);

ES6 提供了更接近传统语言的写法，引入了 Class（类）这个概念，作为对象的模板。通过class关键字，可以定义类。
上面的代码 用es6的语法改写

class A {
  constructor(x, y) {
    this.x = x;
    this.y = y;
  }

  toString() {
    return '(' + this.x + ', ' + this.y + ')';
  }
}

上面代码定义了一个“类”，可以看到里面有一个constructor()方法，这就是构造方法，而this关键字则代表实例对象。

下面我们用class简单创建一个实例

class B {
  api() {
    console.log('创建完了');
  }
}

const b = new B();
b.api() // "创建完了"

构造函数的prototype属性，在 ES6 的“类”上面继续存在。事实上，类的所有方法都定义在类的prototype属性上面。

class Point {
  constructor() {
   
  }
  toString() {
   
  }
  number() {
    
  }
}

// 等同于
Point.prototype = {
  constructor() {},
  toString() {},
  number() {},
};

上面代码中，constructor()、toString()number()这三个方法，其实都是定义在Point.prototype上面。

因此，在类的实例上面调用方法，其实就是调用原型上的方法。
