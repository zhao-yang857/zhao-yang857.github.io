Quick Start

Create a new post

$ hexo new "My New Post"

More info: Writing

Run server

$ hexo server

More info: Server

Generate static files

$ hexo generate

More info: Generating

Deploy to remote sites

$ hexo deploy

More info: Deployment

部署注意事项

安装依赖

npm install hexo-cli -g
npm i

启动
hexo server

部署github

接下来需要我们创建一个git秘钥
打开git bash（不要求在哪个目录）

执行： “这里面是你的邮箱”

ssh-keygen -t rsa -C “your_email@youremail.com“

不管出现什么信息，你只需要回车就可以了
成功界面

然后执行

cat ~/.ssh/id_rsa.pub

会输出你的秘钥，我们复制输出信息就行了

然后点击头像，打开个人设置–选择SSH公钥–新增公钥

然后将你复制的秘钥粘贴进去，记得勾选永久有效

下载部署远程的依赖

npm install hexo-deployer-git --save

依次执行

hexo clean
hexo generate
hexo deploy

或者

hexo clean && hexo g && hexo d
